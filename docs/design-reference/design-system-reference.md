# HALO Design System Reference

This document provides visual references for the key sections and components of the HALO website.

---

## Hero Section

The hero section introduces HALO's core value proposition with a duotone visual style.

![Hero Section](d:/HALO/docs/design-reference/hero-section.png)

**Key Elements:**
- Headline: "Enabling agents and entities become **ECONOMIC ACTORS**"
- Duotone image with green/black color scheme
- Split layout: text left, visual right

---

## Introducing HALO Card

The hero card showcases the HALO brand with a particle/3D sphere animation.

![Introducing HALO Card](d:/HALO/docs/design-reference/introducing-halo-card.png)

**Key Elements:**
- Dark card with rounded corners
- FK Raster font for "HALO" wordmark
- Animated particle sphere on the right
- Tagline: "The unified infrastructure layer that connects merchants, agents, and protocols"

---

## Manifesto Section

The manifesto section explains "The Next Commerce Paradigm" with long-form content.

![Manifesto Section](d:/HALO/docs/design-reference/manifesto-section.png)

**Key Elements:**
- FK Raster display heading
- Neue Montreal body text
- Vertical dot navigation on the right
- Blockquote styling with left green border
- Hyperlinks with underline on hover

## Newsletter Section

The newsletter section features a dithered mountain visual and email signup.

![Newsletter Section](d:/HALO/docs/design-reference/newsletter-section.png)

**Key Elements:**
- "STAY UPDATED" heading in FK Raster strikethrough style
- Green "Sign Up" button with dark text
- Dithered mountain illustration (halftone/stipple effect)
- Light cream background (#F4F0E2)

---

## Footer Section

The footer uses a dark theme with the large HALO wordmark as background.

![Footer Section](d:/HALO/docs/design-reference/footer-section.png)

**Key Elements:**
- Black background (#000000)
- Halftone/dithered header transition
- Large HALO wordmark in dark gray (#1A1A1A) as background
- Contact info and social links (X, LinkedIn)
- Sticky header with "Let's Chat!" CTA button

## Duotone Generator

Interactive tool for generating duotone images with dither effects.

![Duotone Generator](d:/HALO/docs/design-reference/chrome-capture-2026-02-05 (6).png)

**Key Elements:**
- Mode switcher: Dark Mode, Light Mode, Agentic Mode
- "Replace Image" dashed upload area
- Dithered image preview
- Component Options: Download, Zoom In, Zoom Out, Remove Dither
- Cream accent buttons with dark text
